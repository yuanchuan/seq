# seq

Generate strings from a sequence.


```javascript
/* https://rosettacode.org/wiki/99_Bottles_of_Beer */

seq(99, 1)`
  #{n} bottles of beer on the wall,
  #{n} bottles of beer.
  Take one down and pass it around,
  #{n - 1} bottles of beer on the wall.
`
```

## Installation

```bash
npm i @yuanchuan/seq
```
